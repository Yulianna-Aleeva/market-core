# Ядро для интернет-магазина

Базовые классы для работы с интернет-магазином.


## Описание

Реализованы классы для хранения товаров и категорий интернет-магазина:
- `Product` — базовый класс товаров с приватной ценой.
- `Smartphone` и `LawnGrass` — классы-наследники `Product` с дополнительными атрибутами.
- `Category` — класс категории с приватным списком товаров и счётчиками.
- `CategoryIterator` — класс-итератор для перебора товаров категории.
- `Order` — класс заказа с одним товаром и итоговой стоимостью.
- `BaseProduct` — абстрактный базовый класс для всех продуктов.
- `BaseEntity` — абстрактный базовый класс для категорий и заказов.
- `PrintMixin` — миксин для печати информации о созданном объекте.
- `ZeroQuantityError` — исключение при попытке добавить товар с нулевым количеством.
- `MSG` — константы сообщений об ошибках и статусах.
 
Функции для загрузки данных из JSON-файла и создания объектов.


### Возможности

- Валидация количества товара при создании (`ValueError` при `quantity=0`).
- Подсчёт среднего ценника товаров категории (`middle_price`) с округлением до копеек.
- Обработка исключений при добавлении товара в категорию (`try/except/else/finally`).
- Итерация по товарам категории через `CategoryIterator`.
- Загрузка данных из JSON и создание объектов через `new_product`.


## Установка

1. Клонируйте репозиторий из GitHub:  
https://github.com/Yulianna-Aleeva/market-core.git


2. Установите зависимости:  
```bash
poetry install
```


## Запуск

Пример вывода:

`python main.py`

Запуск чтения JSON-файла:

`python -m src.utils.read_json`


## Тестирование

`python -m pytest`

Тестирование с покрытием:

`python -m pytest --cov`
